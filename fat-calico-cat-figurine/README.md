# Fat Calico Cat Figurine (Japanese style, eating)

Files:
- `fat-calico-cat-figurine.scad`
- `fat-calico-cat-figurine.stl`

## Print notes (PETG)
- Layer height: 0.16-0.20 mm
- Walls: 3-4
- Infill: 15-20% gyroid
- Supports: **none** (model designed to avoid major overhangs)
- Orientation: as modeled (flat base on bed)

## Suggested size
Current model footprint is ~90 mm diameter base and ~74 mm tall.
Scale in slicer if needed:
- 85% for desk charm
- 120% for centerpiece

## Painting tip for calico look
Use raised patch islands as guides:
- Base: cream/white
- Patches: orange + black
- Bowl: red lacquer style
